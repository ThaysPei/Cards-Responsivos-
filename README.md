Este projeto é Muito simples, mas com ideias de ouro! Ele tem o objetivo de desenvolver habilidades
relacionadas a criação de componentes cards, uso de props para agregar objetos
do data.js e responsividade SEM O USO de @media query.

De modo que a página e seus componentes internos se adaptam a diferentes tamanhos
rapidamente. Foi necessário compreender o uso do display grid:
grid-template-columns: repeat(auto-fit, minmax(260px, 1fr));

Paralelo a isso, desenvolvi um template reutilizável. Um molde prático de um card que
recebe componentes externos essenciais para a criação do card, a sua manutenção futura
e um código que é perfeitamente escalável.

Podendo ser REUTILIZADO, inclusive, em projetos paralelos, sendo o modelo ideal
para produtos como e-ccommerces.

Tecnologias utilizadas :
  REACT.VITE - JAVASCRIPT - CSS PURO.
